Quick Start

Get started in seconds

# Install
pip install defi-agent-toolkit

# Configure
cp .env.example .env
# Edit .env with your RPC endpoints

# Find best swap route
defi-agent swap --from ETH --to USDC --amount 0.1 --chain ethereum

# Scan yield opportunities
defi-agent yield --chains ethereum base --min-apy 5

# Check portfolio
defi-agent portfolio --address 0x... --chains ethereum base arbitrum

# Check gas prices
defi-agent gas --chains ethereum base --speed fast
